# Licence API

Vous et Marketo êtes chacun désigné ici individuellement comme &quot;Parti&quot; et collectivement comme &quot;Parties&quot;.

1. Définitions

&quot;Limites d’API&quot; signifie des restrictions (par exemple, des limites de taux et des limites de simultanéité) à vos appels d’API, comme décrit sur le site web du développeur de Marketo et décrit de manière substantielle à l’adresse : https://developer.adobe.com/marketo-apis/.

&quot;Interface de programmation d’application&quot; ou &quot;API&quot; désigne un ensemble de routines, classes, paramètres de fonction, protocoles, webhooks, bibliothèques associées et autres instructions fournies dans le code source ou le formulaire Code objet.

&quot;Spécification de l’API&quot; signifie une description ou définition écrite des routines, classes, paramètres de fonction, protocoles, webhooks, bibliothèques associées et autres instructions qui permettent à une application logicielle d’interagir avec la solution Marketo, et comme décrit de manière substantielle à l’adresse : https://developer.adobe.com/marketo-apis/.

&quot;Commentaires&quot; signifie des suggestions ou des commentaires que vous avez fournis à Marketo en rapport avec les produits Marketo (par exemple, les API Marketo ou la solution Marketo).

&quot;Forking&quot; désigne l’acte de création d’un ensemble distinct et/ou distinct d’API basé sur, dérivé de ou modifié des API Marketo.

&quot;API Marketo&quot; désigne un ensemble de routines, de classes, de paramètres de fonction, de protocoles, de bibliothèques associées, d’outils et d’autres instructions fournis dans le code source ou le formulaire de code d’objet qui permettent l’accès ou l’interopérabilité avec la solution Marketo par une application logicielle, comme décrit de manière substantielle et rendu disponible à l’adresse : https://developer.adobe.com/marketo-apis/.

&quot;Solution Marketo&quot; désigne un ou plusieurs services mis à votre disposition par l’intermédiaire des API Marketo et décrits de manière substantielle à l’adresse : http://www.marketo.com/software/.

&quot;Code objet&quot; désigne le code source qui a été interprété, compilé ou traduit d’une autre manière dans un formulaire lisible par un ordinateur.

&quot;Application logicielle&quot; désigne un logiciel créé par vous dans un formulaire Code objet ou Code source qui accède à la solution Marketo ou interagit d’une autre manière avec cette dernière via les API Marketo.

&quot;Code source&quot; désigne le code informatique sous une forme lisible par un humain et, en tant que tel, le code informatique qui n’a pas été interprété ou compilé.

&quot;Use&quot; signifie un droit d’exécution ou de reproduction (comme défini sous 17 U.S.C. § 101 et suivants.) ou autre loi applicable sur le droit d&#39;auteur, mais ne comprend aucun droit de sous-licence ou de distribution.

&quot;Vous&quot; ou &quot;Votre&quot; signifie vous, la personne qui a conclu le présent Contrat ou, si vous souscrivez au présent Contrat pour le compte d’une organisation (ou si vous implémentez la spécification d’API et les API Marketo associées pour le compte ou dans un produit détenu ou sous licence par une organisation), cette organisation.
